# SearchBooks

## 프로젝트 설명

SearchBooks는 Naver Search Api를 이용한 책 검색 어플입니다.

제목, 출판사 이름, 저자등으로 책을 검색할 수 있으며, 원하는 책을 즐겨찾기하여 따로 모아서 볼 수 있습니다. 

또한, 책을 클릭하여 해당 책의 자세한 정보 확인할 수 있습니다.
